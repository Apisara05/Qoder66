# Qoder66

ยินดีต้อนรับสู่ repository Qoder66

## เกี่ยวกับโปรเจคนี้

นี่คือ repository ที่มีไว้สำหรับจัดเก็บไฟล์ข้อความอย่างง่าย โดยมีไฟล์หลักดังนี้:

- `hello.txt` - ไฟล์ที่มีข้อความ "สวัสดี" เป็นภาษาไทย
- `world.txt` - ไฟล์ที่มีข้อความ "ชาว" 
- `AGENTS.md` - ไฟล์ที่มีคำแนะนำสำหรับ Qoder ในการทำงานกับ repository นี้

## การใช้งาน

repository นี้ไม่มีโค้ดหรือคำสั่งพิเศษใด ๆ เพียงเป็นที่เก็บไฟล์ข้อความอย่างง่าย

## การพัฒนา

ไม่จำเป็นต้องมีเครื่องมือพิเศษใด ๆ สำหรับการพัฒนาใน repository นี้
